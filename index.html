OData API Aggregation Logic

Daily Revenue = GET /sap/opu/odata/sap/SD_SALES → filter(Day=X, Round=Y) → SUM(NetValue)
Gross Margin % = ( SUM(Value) − SUM(Cost) ) ÷ SUM(Value) × 100 — Cost sourced from MM module COGS field
Avg Selling Price = SUM(Value) ÷ SUM(Quantity) — transaction-level price rolled up through pipeline
Note: SAP standard reports show static per-round snapshots. This dashboard aggregates raw OData transactions to expose intra-round daily trends not visible in standard ERP views.
